Addendum: In the HTML file, I created the structure for a simple Pig Game application. It includes sections for two players, their scores, and a dice roll button. Additionally, I added a header for the game title and a footer for copyright information.

In the CSS file, I styled the elements to create a visually appealing layout. This included setting up flexbox for the main container, styling player sections, buttons, and the dice image. I also added styles for the header and footer to position them at the top and bottom of the page respectively.

In the JavaScript file, I implemented the game logic. This involved setting up variables to track scores and the active player, creating functions for rolling the dice, holding scores, and switching players based on game rules. Additionally, I added event listeners to handle user interactions with the buttons.
